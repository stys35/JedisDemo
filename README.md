<h1>Jedis Demo</h1>
这是一个使用SpringBoot集成Jedis来对Redis进行操作的项目．使用Jedis的RedisTemplate进行操作．

<h2>项目工具</h2>
<li>
    编译器：IDEA
</li>
<li>
    项目构建工具：Maven
</li>
<li>
    Redis : 4.0.9
</li>
<li>
    Spring Boot : 2.0.3
</li>
<li>
    spring-data-redis :2.0.8
</li>
